hospital09

总算把鉴权的功能实现了，总结一下。

整体的功能就是，所有的请求都需要登陆，这里的请求分为两大种，

一种是最正常的通过网关然后访问的请求，这种请求通过网关的global的filter,需要检验是否有token以及token是都合法，因为网关不依赖common模块，所以这里需要在网关层复制一个common模块的JWTUtils,注意里面的密钥key需要一致，这里的global的filter的白名单是认证服务的url。登陆请求通过网关转发，在认证服务中，因为这里普通用户和医生的用户认证采取的是同一套系统，所以通过前端在传过来的username前加上doctor或者user的识别，如：doctor-张三就表示张三是个医生，所以就应该在authentica里面的loadByuserName就该查询doctor表，如果是user-李四，就表示李四是个普通用户，就该在认证的时候查询user表，我并没有重新在认证服务模块中再复制一份usermapper以及user的实体，而是这里用了openfeign远程调用，所以这里就写了两个client,查询user和doctor的表。如果认证成功，因为我这里有接口鉴权的功能，所以我这里采取的是RBAC权限设计，认证成功之后，还查询了对应用户的权限，并封装在userDetails,然后将其存入到redis中，这里有一个问题就是查询权限的时候，因为在用户对应角色和医生对应角色在一张表中，所以不好区分谁是用户谁是医生，所以我这里就以20000以内为医生的id,，用户的id从20001开始，就可以区分了，这个时候认证流程就结束了。

然后就是当访问到真正的服务时需要鉴权，因为我把普通用户模块和doctor模块都放在了user模块会出现什么问题呢

​ 1.不能在网关层区分普通用户和doctor的权限，所以我并没有在网关层设置服务层面的鉴权，而是鉴别当前用户是否有访问该接口的权限，这里用的是@preAuthor，这个注解可以自己定义逻辑，

查询permission因为我们这里是全部服务都需要登陆，所以我直接把jwtFilter放在了common模块，jwtfliter做了什么呢，就是每个请求过来的时候，就通过token,解析token然后去redis中获取该对象userDetalis,然后将其注入到securityCoontext中，然后在接口上写上@PreAuthorize(“hasAnyAuthority(‘system:user:add’)”)最根本的用法（底层就是去securityCoontext拿到对象的权限比对），然后就可以实现接口层的鉴权。这里需要注意几个点，因为把jwtFilter放在了common模块，而且在jwtFilter模块中将对象userDetails写入在了securityContext中，所以这里需要在common模块中引入Spring securtiy的依赖，那么会出现什么问题，

1.那么相当于所有引入了common的模块都引入了spring security,那又因为spring security引入后默认就会给所有的接口需要authtien，所以就需要在每个模块下都写一个security的config,用来放开所有接口的权限。

2.因为登陆的时候需要openFeign远程调用该模块，所以需要在jwtFilter下设置白名单放行feign调用的接口

真正思考过的人就会问，为什么在网关层已经token认证了还需要在common模块设置一个token的非法检验呢，这是为了防止直接访问服务不通过网关访问时，也要检验token

一：网关层的glabolFilter


public class TokenFilter implements GlobalFilter, Ordered {
    private static final Log log = LogFactory.getLog(GatewayFilter.class);
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String value = exchange.getRequest().getPath().value();
        System.out.println(value);
        // 判断是否符合白名单/api/security-auth/login
        if (value.equals("/api/security-auth/login")) {
            return chain.filter(exchange);
        }
        String token = exchange.getRequest().getHeaders().getFirst("token");
        String check=null;
        log.info(token+"    "+check);
        try {
            //解析token
            check= String.valueOf(JwtUtil.parseJWT(token).getSubject());
        } catch (Exception e) {
            //token解析出错
            exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
            return exchange.getResponse().setComplete();
        }
        //token解析成功但是出错
        if (!(check.contains("doctor-")|| check.contains("user-"))) {
            exchange.getResponse().setStatusCode(HttpStatus.UNAUTHORIZED);
            return exchange.getResponse().setComplete();
        }

        return chain.filter(exchange);

    }

    @Override
    public int getOrder() {
        return -100;
    }
}

实现userDetalsService中的loadUserByUsername方法：

@Service
public class UserDetailsServiceImpl implements UserDetailsService {

    @Autowired
    private UserClient userClient;

    @Autowired
    private IMenuService menuService;

    @Override
    public UserDetails loadUserByUsername(String username) throws UsernameNotFoundException {
        UserVO userVO=new UserVO();
        //根据username前缀的不同来分别进行远程调用，就是查表
        if (username.contains("doctor-")){
            String[] userNameArray =  username.split("-");
            String realUserName=userNameArray[1];
            DoctorFeignVO doctorByFeign = userClient.getDoctorByFeign(realUserName);
            if (Objects.isNull(doctorByFeign)){
                throw new RuntimeException("用户不存在");
            }
            userVO = BeanCopyUtils.copyBean(doctorByFeign, UserVO.class);
        } else if (username.contains("user-")) {
            String[] userNameArray =  username.split("-");
            String realUserName=userNameArray[1];
            UserFeignVO userByFeign = userClient.getUserByFeign(realUserName);
            if (Objects.isNull(userByFeign)){
                throw new RuntimeException("用户不存在");
            }
            userVO = BeanCopyUtils.copyBean(userByFeign, UserVO.class);
        }
        //查询该用户的权限，就是这里20000以内表示doctor,20001开始就是用户
        List<String> permissions = menuService.getPermissions(userVO.getId());

        return new UserDetailsImpl(userVO,permissions);

    }
}

三：common新增的代码

JwtAuthenticationTokenFilter:

@Component
public class JwtAuthenticationTokenFilter extends OncePerRequestFilter {

    @Autowired
    private RedisCache redisCache;
    @Override
    protected void doFilterInternal(HttpServletRequest httpServletRequest, HttpServletResponse httpServletResponse, FilterChain filterChain) throws ServletException, IOException {
        System.out.println(httpServletRequest.getRequestURI());
        //白名单：login和fegin接口
        if (Objects.equals(httpServletRequest.getRequestURI(), "/security-auth/login") ||
                Objects.equals(httpServletRequest.getRequestURI(), "/service-user/doctor/getDoctorByFeign")
               || Objects.equals(httpServletRequest.getRequestURI(), "/service-user/doctor/FeignGetDoctorInfo")
                || Objects.equals(httpServletRequest.getRequestURI(), "/service-user/patient/FeignGetPatientInfo")
                ||  Objects.equals(httpServletRequest.getRequestURI(), "/service-user/user/getUserByFeigns")
                ||  Objects.equals(httpServletRequest.getRequestURI(), "/service-product/product/FeignGetProductInfo")
        ){
            filterChain.doFilter(httpServletRequest,httpServletResponse);
            return;
        }
        //获取请求token
        String token = httpServletRequest.getHeader("token");
        String userID;
        try {
            userID= String.valueOf(JwtUtil.parseJWT(token).getSubject());
        } catch (Exception e) {
            throw new SystemException(AppHttpCodeEnum.NO_OPERATOR_AUTH);
        }
        String[] userIdArray =  userID.split("-");
        String realUserId=userIdArray[1];
        //userIDVO="user-"+userID;
        //redisCache.setCacheObject("Login:"+userIDVO,userDetailsimpl);
        UserDetailsImpl userDetails =null;
        //通过redis获取登陆的时候存的UserDetailsImpl
        if (userID.contains("doctor-")){
            userDetails= redisCache.getCacheObject("Login:" + "doctor-"+realUserId);
        } else if (userID.contains("user-")) {
            userDetails=redisCache.getCacheObject("Login:" + "user-"+realUserId);
        }
        if (Objects.isNull(userDetails)){
            //redis中缓存过期，登陆超时
            ResponseResult responseResult=ResponseResult.errorResult(AppHttpCodeEnum.NEED_LOGIN);
            WebUtils.renderString(httpServletResponse,JSON.toJSONString(responseResult));
            return;
        }
        //将获取的UserDetailsImpl通过UsernamePasswordAuthenticationToken包装存入SecurityContext中
        UsernamePasswordAuthenticationToken usernamePasswordAuthenticationToken=new UsernamePasswordAuthenticationToken(userDetails,null,userDetails.getAuthorities());
        SecurityContextHolder.getContext().setAuthentication(usernamePasswordAuthenticationToken);
        filterChain.doFilter(httpServletRequest,httpServletResponse);

    }
}
