1.去重：select distinct prod_id

2.排序：DESC 降序，默认升序

select cust_id,order_num
from Orders
order by
     cust_id,
     order_num DESC

3.拼接字符串：

select cust_id,cust_name,
upper(concat(substring(cust_name,1,2),substring(cust_city,1,3))) as user_login
from Customers

4.where后面不能直接加聚合函数（分组函数），我们也可以这样记忆：where后面只能跟表中存在的字段。

having 不能单独使用，必须跟 group by 联合使用。

5.子查询

select cust_id
from Orders as a
where a.order_num in(
    select order_num
    from OrderItems
    where item_price>=10
)

6.子查询+group by +having 过滤+where 过滤

select a.id,a.class,a.name,a.score
    from A as a right join (
        select A.class as oneclass,max(score) as maxScore
        from A
        group by class
    )as b on a.class=b.oneclass and a.score=b.maxScore
